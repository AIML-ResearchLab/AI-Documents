# 04 - Data Quality Profiling and Validation

## 4.1 Data Quality Dimensions
- completeness
- validity
- consistency
- timeliness
- uniqueness

## 4.2 Profiling Baseline
Compute:
- null percentages
- type conformity
- value ranges
- distinct counts

## 4.3 Validation Rules
- schema checks
- business rule checks
- relational integrity checks

## 4.4 Quality Gates
Block downstream processing when critical quality thresholds fail.

## 4.5 Real-Time Example
Stop training pipeline if target label missing rate exceeds 2%.
