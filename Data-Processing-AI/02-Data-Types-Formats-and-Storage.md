# 02 - Data Types, Formats, and Storage

## 2.1 Data Modalities
- structured (tables)
- semi-structured (JSON, logs)
- unstructured (text, image, audio, PDF)

## 2.2 File Formats
- CSV: simple, large overhead
- JSON: nested, flexible
- Parquet: columnar and efficient
- Avro/ORC: schema-based storage

## 2.3 Storage Layers
- object storage
- warehouse/lakehouse
- low-latency serving stores

## 2.4 Partitioning and Indexing
Design partition keys for query and processing efficiency.

## 2.5 Practical Rule
Prefer columnar format for large analytical datasets and feature generation jobs.
