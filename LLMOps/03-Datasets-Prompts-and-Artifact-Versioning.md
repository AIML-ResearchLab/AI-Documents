# 03 - Datasets, Prompts, and Artifact Versioning

## 3.1 Artifact Taxonomy
Track datasets, prompts, model configs, retrieval configs, and evaluation assets.

## 3.2 Version Strategy
Use immutable versions and explicit promotion states.

## 3.3 Lineage Tracking
Link outputs to exact data/prompt/model versions for auditability.

## 3.4 Governance
Require review gates before promoting high-impact artifacts.

## 3.5 Real-Time Example
Bug in customer responses is traced to one prompt version and rolled back quickly.
