# 10 - CI/CD for LLM Systems

## 10.1 Pipeline Design
Automate linting, tests, evals, security checks, and deployment gates.

## 10.2 Artifact Promotion
Promote only signed and validated artifacts through environments.

## 10.3 Deployment Safety
Use canary and blue/green rollout patterns.

## 10.4 Rollback Strategy
Keep fast rollback path with preserved prior artifacts.

## 10.5 Real-Time Example
Pipeline blocks release after eval gate detects unacceptable hallucination increase.
