# 08 - PromptOps and Configuration Management

## 8.1 PromptOps Scope
Treat prompts and context templates as governed production artifacts.

## 8.2 Configuration Layers
Separate static defaults, environment overrides, and runtime controls.

## 8.3 Change Governance
Require reviews, tests, and approvals before prompt promotion.

## 8.4 Drift Prevention
Monitor prompt behavior changes with regression suites.

## 8.5 Real-Time Example
Prompt update passes staging tests, then is rolled out to 10% traffic first.
