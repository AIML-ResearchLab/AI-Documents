# 11 - Time Series and Sequential Data Statistics

## 11.1 Components
- Trend
- Seasonality
- Residual noise

## 11.2 Stationarity
Many statistical methods assume stable mean/variance over time.

## 11.3 Autocorrelation
Past values can influence future values; quantify using ACF/PACF intuition.

## 11.4 Rolling Statistics
Use moving averages and rolling variance for monitoring drift and volatility.

## 11.5 Forecast Evaluation
- MAE
- RMSE
- MAPE (careful with near-zero values)

## 11.6 Data Leakage in Time Series
Never use future information during feature construction.

## 11.7 Real-Time Example
Forecast daily support ticket volume and set staffing alerts with prediction intervals.
