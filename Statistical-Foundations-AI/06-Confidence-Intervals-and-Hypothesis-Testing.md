# 06 - Confidence Intervals and Hypothesis Testing

## 6.1 Confidence Intervals
A confidence interval gives a plausible range for a population parameter.

Interpretation: 95% CI does not mean a 95% chance true parameter is inside after seeing data; it refers to long-run procedure coverage.

## 6.2 Hypothesis Testing Workflow
1. State null and alternative hypotheses
2. Choose test statistic
3. Compute p-value
4. Compare with significance level alpha
5. Make decision and report effect size

## 6.3 Type I and Type II Errors
- Type I: false positive
- Type II: false negative

## 6.4 Statistical Power
Probability of detecting true effect when it exists.

## 6.5 Common Tests
- z-test, t-test
- chi-square test
- proportion tests
- nonparametric alternatives

## 6.6 p-Value Misuse Warnings
- Not probability null is true
- Not effect size
- Not practical significance

## 6.7 AI Example
Compare old vs new model conversion uplift with two-proportion z-test and report CI for uplift.
