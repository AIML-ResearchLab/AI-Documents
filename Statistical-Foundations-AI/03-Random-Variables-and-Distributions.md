# 03 - Random Variables and Distributions

## 3.1 Random Variables
- Discrete: countable outcomes
- Continuous: uncountable outcomes

## 3.2 Distribution Functions
- PMF for discrete RVs
- PDF for continuous RVs
- CDF for both

## 3.3 Expectation and Variance
- `E[X]` average long-run value
- `Var(X)` spread around mean

## 3.4 Important Discrete Distributions
- Bernoulli: binary outcomes
- Binomial: number of successes in n trials
- Poisson: count of events in interval
- Geometric: trials until first success

## 3.5 Important Continuous Distributions
- Uniform
- Normal
- Exponential
- Log-normal

## 3.6 Joint and Marginal Distributions
Understand how variables interact and how to isolate one variable from a joint model.

## 3.7 AI Use Cases
- Poisson for event counts
- Normal approximations for errors
- Bernoulli/Binomial for click and conversion modeling
