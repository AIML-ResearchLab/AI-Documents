# 01 - Program Overview and Math Prerequisites

## 1.1 Goal
Build statistical intuition and practical skill to design, evaluate, and trust AI systems.

## 1.2 Why Statistics Matters in AI
- Models learn patterns under uncertainty
- Metrics and validation are statistical decisions
- Deployment quality depends on distributional assumptions

## 1.3 Learning Path
1. Probability and random variables
2. Estimation and testing
3. Bayesian updates and decision making
4. Model diagnostics and experimental design
5. Robustness and causal reasoning

## 1.4 Required Math Basics
- Algebra and functions
- Basic calculus intuition (rates, areas)
- Matrix basics (vectors, dot product)

## 1.5 Notation Quick Reference
- `P(A)` probability of event A
- `E[X]` expectation
- `Var(X)` variance
- `Cov(X, Y)` covariance
- `N(mu, sigma^2)` normal distribution

## 1.6 AI-Relevant Statistical Questions
- Is this model improvement real or noise?
- Is this feature truly predictive or leakage?
- Is metric drop due to drift or random fluctuation?

## 1.7 Baseline Reproducibility Rules
- Fix random seeds
- Log data and model versions
- Report uncertainty with every key metric
