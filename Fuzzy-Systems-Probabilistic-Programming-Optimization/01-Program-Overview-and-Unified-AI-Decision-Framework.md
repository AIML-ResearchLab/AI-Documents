# 01 - Program Overview and Unified AI Decision Framework

## 1.1 Goal
Build practical expertise to model uncertainty and optimize decisions using fuzzy logic, probabilistic programming, and optimization.

## 1.2 Unified Pipeline
1. represent uncertainty
2. infer latent state
3. choose decision policy
4. optimize objective under constraints
5. monitor and iterate

## 1.3 Why This Combination
- fuzzy systems capture linguistic ambiguity
- probabilistic models capture stochastic uncertainty
- optimization converts model outputs into actionable decisions
