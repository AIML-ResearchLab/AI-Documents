# 13 - Hybrid Fuzzy-Probabilistic-Optimization Pipelines

## 13.1 Hybrid Architecture
- fuzzy layer for interpretable expert rules
- probabilistic layer for uncertainty estimation
- optimization layer for action selection

## 13.2 Benefits
Combines interpretability, uncertainty awareness, and decision optimality.

## 13.3 Integration Challenges
Model handoff consistency, calibration mismatch, and latency management.

## 13.4 Real-Time Example
Risk triage engine uses fuzzy severity + Bayesian confidence + constrained optimization.
