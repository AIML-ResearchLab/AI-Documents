# 04 - Defuzzification and Fuzzy Control Basics

## 4.1 Defuzzification
Convert fuzzy output set into crisp action.

## 4.2 Methods
- centroid
- mean of maxima
- weighted average (Sugeno style)

## 4.3 Fuzzy Control Loop
sense -> fuzzify -> infer -> defuzzify -> act.

## 4.4 Real-Time Example
HVAC controller adjusts fan speed from fuzzy comfort and occupancy signals.
