# 08 - Probabilistic Programming Concepts and Workflows

## 8.1 PPL Idea
Specify probabilistic models as code and run automated inference.

## 8.2 Workflow
model definition -> prior selection -> inference -> posterior diagnostics.

## 8.3 Typical Outputs
Posterior samples, credible intervals, and predictive uncertainty.

## 8.4 Real-Time Example
Bayesian demand model implemented in probabilistic programming framework.
