# 14 - Labs, Projects, and Real-Time Examples

## Lab 1: Robust ASR Baseline
- Build ASR pipeline on noisy + clean subsets
- Report WER/CER by accent and channel
- Add phrase hints and compare improvements

## Lab 2: Diarization and Turn Analytics
- Run diarization on two-speaker and overlap-heavy calls
- Compute DER and derive interruption metrics
- Validate coaching insights on a sampled QA set

## Lab 3: Intent + Compliance in Streaming
- Build low-latency keyword + intent detector
- Define p95 latency and precision targets
- Trigger alert events to a mock agent desktop

## Lab 4: Affect and Escalation Detection
- Combine transcript sentiment + prosodic cues
- Calibrate threshold for false positive control
- Build escalation dashboard with confidence bands

## Capstone Mini-Project
Design a multilingual, real-time contact-center analytics pipeline with measurable business KPIs, governance controls, and rollback playbooks.
