# 01 - Program Overview and Linear Algebra for AI

## 1.1 Goal
Build linear algebra intuition and implementation skills required for modern AI models.

## 1.2 Why It Matters
AI operations are mostly linear algebra:
- feature transformations
- neural network layers
- optimization steps
- embedding similarity search

## 1.3 Learning Path
1. Vector and matrix fundamentals
2. Solvers and decompositions
3. Spectral and dimensionality methods
4. Advanced numerical and large-scale AI systems

## 1.4 Outcomes
- understand model math
- debug numerical issues
- optimize matrix-heavy pipelines
