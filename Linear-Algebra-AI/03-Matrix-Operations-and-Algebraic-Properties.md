# 03 - Matrix Operations and Algebraic Properties

## 3.1 Core Operations
- addition/subtraction
- scalar multiplication
- matrix multiplication
- transpose

## 3.2 Multiplication Rules
If `A` is `(m x n)` and `B` is `(n x p)`, `AB` is `(m x p)`.

## 3.3 Key Properties
- associativity: `(AB)C = A(BC)`
- distributivity: `A(B+C)=AB+AC`
- generally not commutative: `AB != BA`

## 3.4 Identity and Zero Matrices
Use identity `I` and zero matrix for algebraic simplification.
