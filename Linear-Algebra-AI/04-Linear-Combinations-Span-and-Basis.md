# 04 - Linear Combinations, Span, and Basis

## 4.1 Linear Combination
A vector formed by weighted sum of vectors.

## 4.2 Span
Set of all vectors reachable by linear combinations of a set.

## 4.3 Linear Independence
Vectors are independent if none can be written as combination of others.

## 4.4 Basis
A minimal independent set spanning a vector space.

## 4.5 Dimension
Number of basis vectors.

## 4.6 AI Relevance
Feature representation quality depends on basis and dimensional structure.
