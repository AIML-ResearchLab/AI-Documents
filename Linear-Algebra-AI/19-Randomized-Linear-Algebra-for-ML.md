# 19 - Randomized Linear Algebra for ML

## 19.1 Motivation
Approximate methods reduce cost on large matrices.

## 19.2 Core Techniques
- random projections
- sketching
- randomized SVD

## 19.3 Guarantees
Randomized methods can preserve geometry with controllable error bounds.

## 19.4 AI Use Cases
- fast approximate nearest neighbors
- low-rank approximation at scale
- large covariance approximations

## 19.5 Real-Time Example
Use randomized SVD to compress massive user-item matrix quickly.
