# 06 - Rank, Nullity, and Linear Independence

## 6.1 Rank
Number of independent columns (or rows) in matrix `A`.

## 6.2 Null Space
Set of vectors `x` such that `Ax = 0`.

## 6.3 Nullity
Dimension of null space.

## 6.4 Rank-Nullity Theorem
For `A` with `n` columns:
`rank(A) + nullity(A) = n`

## 6.5 Practical Impact
Low rank often indicates redundant features or compressible representations.
