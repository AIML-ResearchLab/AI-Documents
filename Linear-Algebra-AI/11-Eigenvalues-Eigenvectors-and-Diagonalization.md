# 11 - Eigenvalues, Eigenvectors, and Diagonalization

## 11.1 Definition
For matrix `A`, eigenpair satisfies:
`Av = lambda v`

## 11.2 Interpretation
Eigenvectors are invariant directions; eigenvalues are scaling factors.

## 11.3 Diagonalization
If enough independent eigenvectors exist:
`A = PDP^{-1}`

## 11.4 AI Uses
- PCA
- spectral clustering
- stability analysis

## 11.5 Practical Note
Symmetric matrices have real eigenvalues and orthogonal eigenvectors.
