# 05 - Linear Systems and Gaussian Elimination

## 5.1 System Form
`Ax = b` represents multiple linear equations.

## 5.2 Row Operations
- swap rows
- scale row
- add multiple of one row to another

## 5.3 Row Echelon Forms
Use elimination to reach upper-triangular or reduced row echelon form.

## 5.4 Solution Cases
- unique solution
- infinitely many solutions
- no solution

## 5.5 AI Relevance
Linear solvers appear in least squares and many optimization subproblems.
