# 02 - Scalars, Vectors, Matrices, and Tensors

## 2.1 Core Objects
- scalar: single value
- vector: ordered 1D values
- matrix: 2D array
- tensor: n-dimensional array

## 2.2 Shapes and Dimensions
Always track shape. Most bugs in AI pipelines are shape mismatches.

## 2.3 Indexing Notation
- vector: `x_i`
- matrix: `A_{ij}`
- tensor: `T_{ijk...}`

## 2.4 Practical Rule
Write expected input and output shapes for every model component.
