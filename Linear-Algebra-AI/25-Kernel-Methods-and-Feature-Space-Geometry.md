# 25 - Kernel Methods and Feature Space Geometry

## 25.1 Kernel Idea
Compute inner products in implicit high-dimensional feature spaces.

## 25.2 Kernel Matrix
Gram matrix `K` must be PSD for valid kernel methods.

## 25.3 Common Kernels
- linear
- polynomial
- RBF

## 25.4 Scalability Challenge
Kernel methods scale poorly with data size; approximations are often required.

## 25.5 AI Relevance
Kernel geometry offers strong non-linear decision boundaries in moderate-size datasets.
