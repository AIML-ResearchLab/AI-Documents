# 09 - Norms, Distances, and Similarity

## 9.1 Common Norms
- L1 norm
- L2 norm
- infinity norm

## 9.2 Distance Metrics
Distance between vectors often uses L2 or L1 depending on task geometry.

## 9.3 Cosine Similarity
Measures angle-based similarity:
`cos(theta) = (x . y) / (||x|| ||y||)`

## 9.4 AI Applications
- nearest-neighbor retrieval
- embedding search
- clustering

## 9.5 Practical Rule
Normalize vectors before cosine-based retrieval.
