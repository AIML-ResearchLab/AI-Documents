# 29 - Monitoring, Drift, and Incident Response for GenAI

## 29.1 Monitoring Signals
quality drift, safety violations, latency regressions, and cost anomalies.

## 29.2 Drift Types
prompt drift, retrieval drift, behavior drift after model updates.

## 29.3 Incident Workflow
detect -> contain -> rollback/fallback -> patch -> postmortem.

## 29.4 Real-Time Example
Fallback to prior prompt version after spike in schema validation failures.
