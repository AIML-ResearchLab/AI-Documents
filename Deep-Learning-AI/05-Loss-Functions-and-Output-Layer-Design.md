# 05 - Loss Functions and Output Layer Design

## 5.1 Loss Selection
- MSE for regression
- cross-entropy for classification
- task-specific custom losses when needed

## 5.2 Output Layers
Match output activation to task:
- linear for regression
- sigmoid for binary classification
- softmax for multiclass

## 5.3 Practical Rule
Misaligned output and loss causes unstable learning and poor calibration.
