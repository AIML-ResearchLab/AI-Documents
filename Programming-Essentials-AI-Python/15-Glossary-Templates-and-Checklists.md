# 15 - Glossary, Templates, and Checklists

## 15.1 Glossary
- Feature: measurable input used by model
- Label/Target: value to predict
- Overfitting: model memorizes training noise
- Underfitting: model too simple to capture signal
- Data drift: input distribution shift over time
- Concept drift: target relationship changes
- Inference: generating predictions from trained model

## 15.3 Model Evaluation Checklist
- Dataset split correctly?
- Leakage checked?
- Baseline established?
- Segment-wise error analysis done?
- Metric thresholds defined?

## 15.4 Deployment Checklist
- Model artifact versioned
- Input schema validated
- Output schema validated
- Monitoring dashboards configured
- Rollback plan tested

## 15.5 Code Review Checklist
- Clear function and variable naming
- Type hints present
- Error handling included
- Logging added at critical points
- Tests cover edge cases
