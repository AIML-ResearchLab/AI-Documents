# 10 - Client Primitives: Roots, Sampling, and Elicitation

## 10.1 Roots
Filesystem boundary hints shared by client (`roots/list`, list change notifications).

## 10.2 Sampling
Servers can request model completion through client (`sampling/createMessage`).

## 10.3 Elicitation
Servers request additional user info (`elicitation/create`) in form or URL mode.

## 10.4 Trust Model
Human-in-the-loop controls should be available for sampling and elicitation.
