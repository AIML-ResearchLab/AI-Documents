# 08 - Resources, Templates, Subscriptions, and Metadata

## 8.1 Core Methods
- `resources/list`
- `resources/read`
- `resources/templates/list`
- optional `resources/subscribe`

## 8.2 Notifications
`notifications/resources/list_changed` and `notifications/resources/updated`.

## 8.3 Metadata
Use URI, MIME type, size, and annotations for prioritization and audience hints.

## 8.4 Practical Rule
Design stable URI schemes and clear source provenance.
