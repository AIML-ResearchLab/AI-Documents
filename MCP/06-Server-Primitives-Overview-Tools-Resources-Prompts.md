# 06 - Server Primitives Overview: Tools, Resources, Prompts

## 6.1 Tools
Executable actions models can invoke.

## 6.2 Resources
Context data exposed via URI-based access.

## 6.3 Prompts
Reusable user-controlled templates with arguments.

## 6.4 Discovery Model
Clients discover primitives via `*/list` operations and then invoke/read/get as needed.
