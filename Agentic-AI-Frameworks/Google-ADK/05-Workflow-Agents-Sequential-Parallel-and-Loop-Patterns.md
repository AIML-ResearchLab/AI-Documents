# 05 - Workflow Agents: Sequential, Parallel, and Loop Patterns

## 5.1 Sequential Pattern
Deterministic step-by-step execution for auditable workflows.

## 5.2 Parallel Pattern
Execute independent tasks concurrently to reduce total latency.

## 5.3 Loop Pattern
Iterative workflows for refinement, retries, or bounded convergence.

## 5.4 Pattern Selection
Pick orchestration style based on risk, complexity, and throughput needs.

## 5.5 Real-Time Example
Claims processing runs checks in parallel, then loops remediation until validation passes.
