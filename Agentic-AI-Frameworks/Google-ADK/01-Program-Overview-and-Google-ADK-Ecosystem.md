# 01 - Program Overview and Google ADK Ecosystem

## 1.1 Goal
Build complete production-ready expertise in Google Agent Development Kit (ADK).

## 1.2 ADK Positioning
ADK is a modular, model-agnostic framework for building, orchestrating, and deploying AI agents.

## 1.3 Ecosystem Components
Agents, tools, sessions, memory, runtime, deployment, observability, and evaluation.

## 1.4 Learning Path
Foundations -> orchestration -> enterprise controls -> production operations.

## 1.5 Real-Time Example
Customer-support system combines multi-agent delegation with tool-backed policy retrieval.
