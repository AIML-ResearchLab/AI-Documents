# 03 - Data Foundations: Documents, Nodes, and Indexes

## 3.1 Document Model
Documents represent raw source data entering the pipeline.

## 3.2 Node Abstraction
Nodes are transformed chunks that carry text, metadata, and relationships.

## 3.3 Index Role
Indexes organize nodes for efficient retrieval under different query patterns.

## 3.4 Metadata Strategy
Metadata quality strongly impacts filtering and answer grounding.

## 3.5 Real-Time Example
Legal corpus is split into metadata-rich nodes enabling jurisdiction-aware retrieval.
