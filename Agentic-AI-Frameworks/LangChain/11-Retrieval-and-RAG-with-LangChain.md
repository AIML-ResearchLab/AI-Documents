# 11 - Retrieval and RAG with LangChain

## 11.1 Retrieval Purpose
Retrieval adds external, current knowledge to overcome model context and freshness limits.

## 11.2 RAG Architectures
2-step RAG, agentic RAG, and hybrid RAG each balance control and flexibility differently.

## 11.3 Building Blocks
Loaders, splitters, embeddings, vector stores, retrievers, and reranking pipelines.

## 11.4 Production Criteria
Evaluate recall, citation quality, latency, and answer faithfulness.

## 11.5 Real-Time Example
Internal knowledge agent switches between direct answer and retrieval-driven response by intent.
