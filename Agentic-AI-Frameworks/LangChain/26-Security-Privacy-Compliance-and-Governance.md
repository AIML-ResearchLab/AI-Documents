# 26 - Security, Privacy, Compliance, and Governance

## 26.1 Security Layers
Protect prompts, memory stores, tool endpoints, and model credentials.

## 26.2 Privacy Controls
Implement PII redaction, retention limits, and data minimization.

## 26.3 Compliance Framework
Align agent behavior with legal and internal policy requirements.

## 26.4 Governance Model
Assign ownership for prompts, tools, evaluations, and release approvals.

## 26.5 Real-Time Example
Healthcare agent enforces PHI masking and audit logging before external API calls.
