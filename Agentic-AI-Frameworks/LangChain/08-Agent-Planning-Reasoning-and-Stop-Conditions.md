# 08 - Agent Planning, Reasoning, and Stop Conditions

## 8.1 Planning Behavior
Agents decompose tasks into iterative model reasoning and tool action steps.

## 8.2 Control Parameters
Set iteration caps, timeout budgets, and fallback flows.

## 8.3 Determinism vs Flexibility
Balance autonomous exploration with strict workflow constraints.

## 8.4 Termination Logic
Stop safely when goals are met or uncertainty exceeds policy threshold.

## 8.5 Real-Time Example
Procurement agent stops early and requests approval when quote confidence is insufficient.
