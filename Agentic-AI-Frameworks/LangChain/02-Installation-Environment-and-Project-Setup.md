# 02 - Installation, Environment, and Project Setup

## 2.1 Core Packages
Install `langchain` plus provider-specific packages such as `langchain-openai` or `langchain-anthropic`.

## 2.2 Environment Design
Use per-project virtual environments, pinned dependencies, and secure API key management.

## 2.3 Dev Workflow
Separate local prototyping, staging validation, and production deployment configurations.

## 2.4 Configuration Hygiene
Centralize model IDs, runtime limits, and tracing flags through environment-driven settings.

## 2.5 Real-Time Example
A team runs the same agent code in dev and prod by swapping environment-specific model settings.
