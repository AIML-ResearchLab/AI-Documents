# 19 - Agentic RAG, Hybrid RAG, and Self-Correction Patterns

## 19.1 Agentic RAG
Agent decides when to retrieve, from where, and how to use evidence.

## 19.2 Hybrid RAG
Blend vector retrieval, lexical retrieval, and structured sources.

## 19.3 Self-Correction
Add critique and verification passes before final answers.

## 19.4 Reliability Controls
Require citations and confidence thresholds for high-stakes responses.

## 19.5 Real-Time Example
Policy agent re-queries missing evidence when citation confidence is too low.
