# 01 - Program Overview and LangChain Ecosystem

## 1.1 Goal
Build complete practical and production-ready expertise in LangChain for agentic AI applications.

## 1.2 LangChain in the Stack
LangChain provides high-level agent abstractions built on top of LangGraph runtime capabilities.

## 1.3 Current Direction
LangChain v1 centers on `create_agent`, standardized content blocks, and streamlined namespaces.

## 1.4 Ecosystem Components
Models, messages, tools, memory, middleware, retrieval, guardrails, and LangSmith operations.

## 1.5 Real-Time Example
Support assistant routes customer queries through model reasoning and tool calls in one runtime loop.
