# 14 - Human-in-the-Loop, Runtime Controls, and Streaming

## 14.1 Human-in-the-Loop Patterns
Insert approval checkpoints for high-risk or high-impact actions.

## 14.2 Runtime Controls
Control execution with interrupts, timeout boundaries, and context-aware gating.

## 14.3 Streaming UX
Stream intermediate events and tool progress for responsive user experiences.

## 14.4 Operational Safety
Escalate unresolved ambiguity to humans instead of forcing autonomous decisions.

## 14.5 Real-Time Example
Finance agent pauses before wire-transfer action and resumes after human approval.
