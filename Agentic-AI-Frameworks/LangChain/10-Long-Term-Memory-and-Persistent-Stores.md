# 10 - Long-Term Memory and Persistent Stores

## 10.1 Persistent Memory Model
Long-term memory stores user or application data across sessions.

## 10.2 Store Patterns
Use namespace/key strategies for tenant-safe and queryable memory access.

## 10.3 Operational Stores
Prefer persistent backends such as Postgres-backed stores for production.

## 10.4 Governance Controls
Apply retention, deletion, and access policies to stored memory artifacts.

## 10.5 Real-Time Example
Customer-success agent recalls account history from previous sessions for personalized support.
