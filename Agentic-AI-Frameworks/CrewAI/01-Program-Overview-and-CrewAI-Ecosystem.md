# 01 - Program Overview and CrewAI Ecosystem

## 1.1 Goal
Build complete practical and production-level expertise in CrewAI for agentic systems.

## 1.2 CrewAI Positioning
CrewAI focuses on collaborative agents, crews, and event-driven flows for production automations.

## 1.3 Core Building Blocks
Agents, Tasks, Crews, Processes, Flows, Memory, Knowledge, Tools, and Event Listeners.

## 1.4 Runtime Philosophy
Composable primitives enable both fast prototyping and hardened production execution.

## 1.5 Real-Time Example
Support automation combines specialist agents and flow routing for triage and escalation.
