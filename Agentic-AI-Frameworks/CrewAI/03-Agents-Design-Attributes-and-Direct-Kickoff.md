# 03 - Agents: Design, Attributes, and Direct Kickoff

## 3.1 Agent Concept
An agent is an autonomous unit with role, goal, tools, memory, and execution constraints.

## 3.2 Agent Attributes
Important controls include `max_iter`, `allow_delegation`, `reasoning`, tool access, and callbacks.

## 3.3 Creation Patterns
Define agents via YAML (recommended) or code-level declarations.

## 3.4 Direct Interaction
`kickoff()` supports direct agent usage without full crew orchestration.

## 3.5 Real-Time Example
Research agent with delegated search tools returns scoped findings for a product brief.
