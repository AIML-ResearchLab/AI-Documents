# 06 - Processes: Sequential, Hierarchical, and Hybrid Orchestration

## 6.1 Sequential Process
Tasks run in deterministic order, ideal for linear and auditable workflows.

## 6.2 Hierarchical Process
Manager agent or manager LLM delegates and validates task execution paths.

## 6.3 Hybrid Control
Mix deterministic sequencing with manager-driven adaptation for complex workflows.

## 6.4 Strategy Selection
Choose process mode by risk profile, variability, and throughput needs.

## 6.5 Real-Time Example
Incident workflow uses hierarchical manager for adaptive escalation under changing severity.
