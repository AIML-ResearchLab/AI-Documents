# 16 - Advanced Agent Architecture and Role Engineering

## 16.1 Role Design
Define narrow, high-signal roles with explicit goals and constraints.

## 16.2 Capability Partitioning
Separate planning, execution, and validation responsibilities across agents.

## 16.3 Behavioral Controls
Tune `reasoning`, iteration limits, and tool scopes for predictable outcomes.

## 16.4 Persona Governance
Version and test role/backstory prompts as controlled artifacts.

## 16.5 Real-Time Example
Risk-analysis crew uses separate investigator and auditor agents to reduce false positives.
