# 21 - Security, Privacy, and Governance in Agentic Systems

## 21.1 Security Boundaries
Protect agent runtime, tools, secrets, and external integrations.

## 21.2 Privacy Controls
Apply PII minimization, masking, retention policies, and tenant isolation.

## 21.3 Governance Framework
Define ownership for prompts, tools, flows, and production change approvals.

## 21.4 Auditability
Capture agent decisions, task outcomes, and human approvals for compliance.

## 21.5 Real-Time Example
Healthcare workflow masks PHI before tool calls and logs all approval checkpoints.
