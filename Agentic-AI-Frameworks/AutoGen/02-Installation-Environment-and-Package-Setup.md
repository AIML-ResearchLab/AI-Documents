# 02 - Installation, Environment, and Package Setup

## 2.1 Package Layout
Use `autogen-agentchat`, `autogen-core`, `autogen-ext`, and optionally `autogenstudio`.

## 2.2 Environment Design
Manage provider keys and runtime settings with isolated environments and secure configuration.

## 2.3 Setup Workflow
Install minimal components first, then add extensions based on tools and runtime needs.

## 2.4 Dependency Hygiene
Pin versions and maintain reproducible lockfiles for stable deployments.

## 2.5 Real-Time Example
Team starts with AgentChat prototype and adds Core/Extensions as workflow complexity grows.
