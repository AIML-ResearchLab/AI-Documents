# 03 - AutoGen Architecture: AgentChat, Core, Extensions, and Studio

## 3.1 Layered Design
AgentChat offers high-level APIs; Core exposes event-driven primitives; Extensions add provider/tool integrations.

## 3.2 Runtime Model
Agents exchange typed messages through runtime-managed communication channels.

## 3.3 Extensibility
Custom agents, model clients, and tools plug into the same orchestration model.

## 3.4 Studio Role
AutoGen Studio enables visual prototyping and iterative workflow design.

## 3.5 Real-Time Example
Developer prototypes in Studio, exports architecture, and hardens runtime logic in code.
