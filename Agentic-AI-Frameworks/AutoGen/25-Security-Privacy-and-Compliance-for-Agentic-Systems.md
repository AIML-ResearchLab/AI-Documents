# 25 - Security, Privacy, and Compliance for Agentic Systems

## 25.1 Security Layers
Secure model credentials, tool access, state stores, and network boundaries.

## 25.2 Privacy Controls
Apply PII minimization, masking, retention controls, and deletion workflows.

## 25.3 Compliance Mapping
Map workflow behavior to internal policy and regulatory requirements.

## 25.4 Continuous Assurance
Run policy checks and audit sampling as part of release governance.

## 25.5 Real-Time Example
Healthcare agent enforces PHI controls and logs all external data access events.
