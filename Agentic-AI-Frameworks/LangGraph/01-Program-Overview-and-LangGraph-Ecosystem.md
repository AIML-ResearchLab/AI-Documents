# 01 - Program Overview and LangGraph Ecosystem

## 1.1 Goal
Build complete production-level expertise in LangGraph for durable agent systems.

## 1.2 Framework Positioning
LangGraph provides low-level graph orchestration for stateful and controllable agent workflows.

## 1.3 Core Components
StateGraph, nodes, edges, reducers, checkpoints, interrupts, and runtime hooks.

## 1.4 Why LangGraph
It enables deterministic control where standard agent loops are insufficient.

## 1.5 Real-Time Example
Support pipeline pauses for approval, resumes, and records state for auditability.
