# 09 - Tool Calling and External System Integration

## 9.1 Tool Contracts
Tools should expose strict schemas and stable response formats.

## 9.2 Integration Patterns
Connect APIs, databases, and internal services through dedicated nodes.

## 9.3 Safety Boundaries
Validate arguments and apply least-privilege authorization for tool use.

## 9.4 Reliability
Add retries and idempotency handling for side-effecting actions.

## 9.5 Real-Time Example
Agent calls CRM and ticket tools to assemble escalation context.
