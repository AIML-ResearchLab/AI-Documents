# 04 - Nodes, Edges, Commands, and Routing

## 4.1 Node Responsibilities
Nodes encapsulate distinct steps such as retrieval, reasoning, or action.

## 4.2 Edge Types
Use static and conditional edges to route execution between nodes.

## 4.3 Commands
Commands can direct graph traversal and control branching behavior.

## 4.4 Routing Design
Router nodes centralize branching logic for maintainable orchestration.

## 4.5 Real-Time Example
Router sends billing issues to finance branch and outages to incident branch.
