# 08 - Memory Strategies: Short-Term and Long-Term

## 8.1 Short-Term Memory
Thread-local memory retains immediate conversational context.

## 8.2 Long-Term Memory
Persistent memory stores user, domain, or workflow knowledge.

## 8.3 Memory Boundaries
Separate session context from durable profile memory.

## 8.4 Lifecycle Policies
Define memory retention, summarization, and deletion workflows.

## 8.5 Real-Time Example
Support assistant recalls account preferences while pruning stale history.
