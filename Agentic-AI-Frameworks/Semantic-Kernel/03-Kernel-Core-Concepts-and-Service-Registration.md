# 03 - Kernel Core Concepts and Service Registration

## 3.1 Kernel Role
Kernel is the orchestration container that coordinates services, functions, and execution context.

## 3.2 Service Registration
Register chat completion, embedding, and other AI services in kernel configuration.

## 3.3 Execution Context
Pass runtime data, user variables, and settings to prompt/function execution.

## 3.4 Composability
Kernel design supports modular addition of plugins, memory, and orchestration logic.

## 3.5 Real-Time Example
A logistics kernel routes delivery prompts through configured provider and routing policies.
