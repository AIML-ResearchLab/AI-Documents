# 05 - Plugins, Native Functions, and OpenAPI Integration

## 5.1 Plugin Concept
Plugins expose domain capabilities to the model through callable functions.

## 5.2 Native vs Semantic Functions
Native functions are code-backed; semantic functions are prompt-backed.

## 5.3 OpenAPI Plugins
Import API operations as callable functions for rapid enterprise integration.

## 5.4 Contract Design
Define stable schemas and side-effect boundaries for safe plugin execution.

## 5.5 Real-Time Example
Billing assistant invokes invoice API plugin and summarizes results in policy-compliant format.
