# 19 - Advanced Function Calling and Planning Strategies

## 19.1 Planning Modes
Use static plans for predictable flows and dynamic plans for exploratory tasks.

## 19.2 Function Selection Controls
Constrain callable functions by risk profile, role, and runtime context.

## 19.3 Loop Governance
Set stop conditions, budget limits, and failure recovery paths.

## 19.4 Validation Layer
Apply post-call verification before downstream state updates.

## 19.5 Real-Time Example
Procurement planner calls pricing, policy, and approval functions under strict execution budget.
