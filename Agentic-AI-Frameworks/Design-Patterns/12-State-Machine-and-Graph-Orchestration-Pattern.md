# 12 - State Machine and Graph Orchestration Pattern

## 12.1 Pattern Intent
Model workflow states explicitly for deterministic control and replayability.

## 12.2 State Design
Define valid transitions, terminal conditions, and error boundaries.

## 12.3 Graph Benefits
Supports branching, loops, subgraphs, and resumability.

## 12.4 Debugging Advantage
Explicit state transitions simplify incident root-cause analysis.

## 12.5 Real-Time Example
Onboarding flow resumes from last approved state after service interruption.
