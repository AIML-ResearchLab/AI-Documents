# 11 - Event-Driven Asynchronous Agent Pattern

## 11.1 Pattern Intent
Decouple agent tasks using event streams and asynchronous processing.

## 11.2 Event Contracts
Define stable event schemas and routing topics.

## 11.3 Reliability Controls
Use retries, dead-letter queues, and replay-safe consumers.

## 11.4 Throughput Benefits
Asynchrony improves scalability for long-running workflows.

## 11.5 Real-Time Example
Ticket events trigger triage agents without blocking user-facing interactions.
