# 24 - Cost-Aware Adaptive Model Routing Pattern

## 24.1 Pattern Intent
Optimize quality-cost balance through dynamic model selection.

## 24.2 Routing Rules
Route by complexity, confidence, latency budget, and compliance tier.

## 24.3 Escalation Logic
Escalate to stronger models only when confidence is low.

## 24.4 Budget Governance
Enforce per-workflow token and spend limits.

## 24.5 Real-Time Example
Support bot handles standard queries with low-cost model and escalates disputes.
