# 20 - Protocol-First Interoperability: MCP and A2A Pattern

## 20.1 Pattern Intent
Standardize interoperability through shared protocol contracts.

## 20.2 MCP Role
MCP provides portable tool/resource integration across hosts.

## 20.3 A2A Role
A2A enables agent-to-agent task delegation across systems.

## 20.4 Security Model
Use scoped permissions, identity checks, and audit logging.

## 20.5 Real-Time Example
Coordinator delegates vendor analysis via A2A and uses MCP for internal data tools.
