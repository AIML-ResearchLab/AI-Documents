# 12 - Layer 12: Memory, Session, and Personalization

Preserve relevant context across interactions while enforcing safety boundaries.

## 12.2 Core Decisions

## 12.3 Design Artifacts
Memory schema, retention policy, and personalization controls.

Unbounded memory creates privacy risk and context noise.
