# 11 - Layer 11: Agent Orchestration and Workflow Control

## 11.1 Purpose
Coordinate planning, delegation, and execution through explicit workflow logic.

## 11.2 Core Decisions
Choose supervisor, planner-executor, or graph-state orchestration patterns.

## 11.3 Design Artifacts
Workflow graph, state contracts, and routing rules.

## 11.4 Failure Mode
Implicit orchestration leads to non-deterministic and hard-to-debug behavior.

## 11.5 Real-Time Example
Incident agent graph routes triage, remediation, and approval branches.
