# 11 - RL Evaluation Metrics and Benchmarking

## 11.1 Core Metrics
- average episodic return
- success rate
- sample efficiency
- regret (where relevant)

## 11.2 Evaluation Protocol
multiple seeds, confidence intervals, and fixed benchmark environments.

## 11.3 Pitfalls
overfitting to simulator, unstable comparisons, and cherry-picked runs.

## 11.4 Reporting Rule
Always report variance and failure cases, not only best run.
