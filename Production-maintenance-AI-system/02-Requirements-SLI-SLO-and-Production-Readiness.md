# 02 - Requirements, SLI/SLO, and Production Readiness

## 2.1 Requirements
Define business objective, user impact, risk tier, and constraints.

## 2.2 Reliability Targets
Track SLIs (latency, error rate, freshness, quality) and set SLOs.

## 2.3 Readiness Checklist
No production go-live without explicit threshold gates and rollback plan.

## 2.4 Real-Time Example
Set p95 latency and quality SLO before serving policy assistant to users.
