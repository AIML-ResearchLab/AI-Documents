# 02 - Embeddings, Vector Geometry, and Representation

## 2.1 Embedding Fundamentals
Embeddings convert unstructured data into numeric vectors in high-dimensional space.

## 2.2 Dense vs Sparse Vectors
Dense vectors capture semantic similarity; sparse vectors capture lexical term-weight patterns.

## 2.3 Vector Dimensionality
Higher dimensions can improve expressiveness but increase storage and latency costs.

## 2.4 Normalization and Consistency
Use consistent models and preprocessing for both document and query vectors.

## 2.5 Real-Time Example
An e-commerce system embeds product titles and user queries with the same model for consistent matching.
