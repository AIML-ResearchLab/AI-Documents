# 29 - Azure AI Search Vector: Architecture, Features, and Operations

## 29.1 Positioning
Azure AI Search provides managed enterprise search with vector and hybrid retrieval capabilities.

## 29.2 Data Model
Indexes documents, metadata, and vectors through ingestion and enrichment pipelines.

## 29.3 Retrieval Features

Strong integration with Azure identity, networking, and enterprise governance controls.
