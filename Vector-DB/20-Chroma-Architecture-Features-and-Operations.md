# 20 - Chroma: Architecture, Features, and Operations

## 20.1 Positioning
Chroma is a developer-friendly embedding database used heavily in prototyping and local AI apps.

## 20.2 Data Model
Organizes vectors into collections with document text, IDs, and metadata.

## 20.3 Retrieval Features
Supports similarity search, filtered retrieval, and simple API integration with AI frameworks.

## 20.4 Operations
Easy local setup accelerates experimentation; production deployments need disciplined operations design.

## 20.5 Best Fit
Rapid prototyping, local assistants, and early-stage product development.

## 20.6 Real-Time Example
Team validates new chunking strategy locally before moving to a managed production database.
