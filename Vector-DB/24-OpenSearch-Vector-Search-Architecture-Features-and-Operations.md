# 24 - OpenSearch Vector Search: Architecture, Features, and Operations

## 24.1 Positioning
OpenSearch provides open-source search and analytics with built-in vector search options.

## 24.2 Data Model
Stores documents with vector fields and filterable metadata in index structures.

## 24.3 Retrieval Features
Supports kNN and hybrid retrieval workflows with configurable query pipelines.

## 24.4 Operations
Cluster-based deployment supports scaling, security controls, and observability integrations.

## 24.5 Best Fit
Organizations preferring open-source search stacks with semantic retrieval support.

## 24.6 Real-Time Example
Ops assistant retrieves incident runbooks via hybrid search over logs and knowledge docs.
