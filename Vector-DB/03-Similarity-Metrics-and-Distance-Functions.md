# 03 - Similarity Metrics and Distance Functions

## 3.1 Common Metrics
cosine similarity, dot product, Euclidean distance, and Manhattan distance.

## 3.2 Metric Selection Rules
Choose metric based on embedding model guidance and normalization behavior.

## 3.3 Ranking Implications
Different metrics change nearest-neighbor ordering and retrieval quality.

## 3.4 Consistency Requirement
Train-time and retrieval-time metric mismatch causes relevance degradation.

## 3.5 Real-Time Example
Normalized text embeddings perform best with cosine distance in a legal search pipeline.
