# 28 - MongoDB Atlas Vector Search: Architecture, Features, and Operations

## 28.1 Positioning
MongoDB Atlas Vector Search brings semantic retrieval into document database workflows.

## 28.2 Data Model
Stores vectors alongside JSON-like documents and operational application data.

## 28.3 Retrieval Features
Supports vector queries in aggregation pipelines with filtering and scoring combinations.

## 28.4 Operations
Managed cloud operations with familiar MongoDB tooling and access controls.

## 28.5 Best Fit
Application teams wanting unified operational + vector data in a single platform.

## 28.6 Real-Time Example
Support app retrieves semantically similar tickets and joins customer profile history.
