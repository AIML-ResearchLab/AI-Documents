# 04 - ANN Algorithms: HNSW, IVF, PQ, and Disk-Based Search

## 4.1 Why ANN
Approximate nearest neighbor search trades small recall loss for large latency gains.

## 4.2 HNSW
Graph-based index with strong recall and low latency for many interactive use cases.

## 4.3 IVF and PQ
Cluster-based partitioning and quantization reduce memory and improve large-scale efficiency.

## 4.4 Disk-Based Strategies
Disk-backed indexes reduce RAM pressure and help with very large collections.

## 4.5 Real-Time Example
Media platform serves millions of recommendations with ANN and tuned recall targets.
