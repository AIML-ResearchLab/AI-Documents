# 10 - Evaluation, Benchmarking, and Recall-Latency Tradeoffs

## 10.1 Core Metrics
Recall@k, NDCG, MRR, p95 latency, throughput, and cost per query.

## 10.2 Offline Evaluation
Use labeled or weakly labeled datasets to compare retrieval variants.

## 10.3 Online Evaluation
Run controlled A/B experiments with quality and business KPIs.

## 10.4 Tradeoff Management
Set explicit SLOs for latency while protecting minimum recall.

## 10.5 Real-Time Example
Team rejects low-latency configuration because citation correctness drops below target.
