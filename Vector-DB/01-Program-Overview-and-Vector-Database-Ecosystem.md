# 01 - Program Overview and Vector Database Ecosystem

## 1.1 Goal
Build complete production-level understanding of vector databases for AI systems.

## 1.2 What Vector Databases Enable
Fast similarity search over embeddings for semantic retrieval, recommendation, personalization, and agent memory.

## 1.3 Core Building Blocks
embeddings -> indexing -> metadata filtering -> retrieval -> reranking -> evaluation -> operations.

## 1.4 Ecosystem Categories
- pure vector databases
- vector-enabled search engines
- vector-enabled transactional databases
- managed cloud vector search services

## 1.5 Real-Time Example
Customer support assistant retrieves policy chunks by meaning instead of keyword match.
