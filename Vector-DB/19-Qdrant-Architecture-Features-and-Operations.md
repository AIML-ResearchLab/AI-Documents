# 19 - Qdrant: Architecture, Features, and Operations

## 19.1 Positioning
Qdrant is an open-source vector database focused on efficient filtering and practical production use.

## 19.2 Data Model
Uses collections of points with vectors plus payload fields for structured filtering.

## 19.3 Retrieval Features
Provides ANN retrieval, payload-aware filtering, and optimization features for performance.

## 19.4 Operations
Supports snapshots, replication, and scaling patterns for self-hosted or managed deployments.

## 19.5 Best Fit
Applications requiring strong filter + vector combinations and open deployment control.

## 19.6 Real-Time Example
Fraud detection assistant retrieves similar historical events filtered by account segment.
