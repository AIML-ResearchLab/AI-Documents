# 17 - Weaviate: Architecture, Features, and Operations

## 17.1 Positioning
Weaviate is an open-source vector database with managed cloud options and rich retrieval features.

## 17.2 Data Model
Schema-driven classes/collections store objects, vectors, and filterable properties.

## 17.3 Retrieval Features
Supports vector search, hybrid lexical-vector retrieval, metadata filters, and reranking integrations.

## 17.4 Operations
Offers clustering, sharding, replication, and multi-tenant patterns for enterprise use.

## 17.5 Best Fit
Teams needing open-source flexibility plus strong hybrid retrieval capabilities.

## 17.6 Real-Time Example
Knowledge platform uses hybrid search to combine keyword precision with semantic recall.
