# 23 - Elasticsearch Vector Search: Architecture, Features, and Operations

## 23.1 Positioning
Elasticsearch combines lexical search, analytics, and vector retrieval in one platform.

## 23.2 Data Model
Indexes documents with structured fields, text analysis pipelines, and vector fields.

## 23.3 Retrieval Features
Supports semantic retrieval, hybrid ranking, metadata filtering, and query DSL composition.

## 23.4 Operations
Mature cluster operations include lifecycle policies, observability, and role-based access control.

## 23.5 Best Fit
Search-heavy systems needing both semantic and traditional full-text capabilities.

## 23.6 Real-Time Example
Documentation portal blends BM25, vector similarity, and freshness in unified ranking.
