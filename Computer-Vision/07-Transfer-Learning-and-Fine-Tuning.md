# 07 - Transfer Learning and Fine-Tuning

## 7.1 Why Transfer Learning
Pretrained backbones reduce data and compute requirements.

## 7.2 Strategies
- feature extraction (freeze backbone)
- partial fine-tuning
- full fine-tuning

## 7.3 Practical Rule
Start with frozen backbone baseline before full-scale tuning.

## 7.4 Real-Time Example
Fine-tune pretrained vision model for defect classification with small dataset.
