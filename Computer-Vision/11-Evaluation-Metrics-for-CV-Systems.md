# 11 - Evaluation Metrics for CV Systems

## 11.1 Classification Metrics
accuracy, precision, recall, F1, calibration.

## 11.2 Detection Metrics
IoU thresholds, AP/mAP, size-specific performance.

## 11.3 Segmentation Metrics
mean IoU, Dice, boundary-aware metrics.

## 11.4 Reporting Rule
Always include class-wise and subgroup metrics.
