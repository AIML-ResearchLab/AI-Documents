# 23 - NebulaGraph: Architecture, Features, and Operations

## 23.1 Positioning
NebulaGraph is an open-source distributed graph database built for large-scale graph datasets.

## 23.2 Data Model and Query
Property graph model with query language support for graph traversals and analytics.

## 23.3 Strengths
Scalable architecture, separation of storage and compute, and high concurrency support.

## 23.4 Operational Notes
Tune partition distribution, query concurrency, and index usage for stable performance.

## 23.5 Best Fit
Massive knowledge graphs and recommendation systems with high query concurrency.

## 23.6 Real-Time Example
E-commerce graph handles large item-user interaction networks for personalized discovery.
