# 09 - Graph Embeddings and Graph Representation Learning

## 9.1 Embedding Goal
Project nodes, edges, or whole graphs into vectors for downstream ML tasks.

## 9.2 Common Techniques
Node2Vec, DeepWalk, GraphSAGE-style methods, and relation-aware embeddings.

## 9.3 Use Cases
Similarity search, recommendation, classification, anomaly detection, and retrieval.

## 9.4 Operational Considerations
Retrain embeddings when topology drifts or relation semantics change.

## 9.5 Real-Time Example
Fraud model flags accounts whose embedding neighborhoods shift abruptly.
