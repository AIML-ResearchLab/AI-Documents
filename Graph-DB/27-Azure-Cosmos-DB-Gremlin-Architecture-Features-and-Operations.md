# 27 - Azure Cosmos DB Gremlin: Architecture, Features, and Operations

## 27.1 Positioning
Azure Cosmos DB provides managed graph capability through Gremlin API.

## 27.2 Data Model and Query
Property graph model with Gremlin traversal for relationship-centric workloads.

## 27.3 Strengths
Global distribution, low-latency access patterns, and Azure ecosystem integration.

## 27.4 Operational Notes
Design partition keys and RU budgets carefully for traversal-heavy query paths.

## 27.5 Best Fit
Globally distributed applications requiring managed graph with cloud-native operations.

## 27.6 Real-Time Example
Global IoT graph correlates device-to-location relationships across multiple regions.
