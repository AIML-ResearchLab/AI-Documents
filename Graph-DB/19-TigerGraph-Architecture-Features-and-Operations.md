# 19 - TigerGraph: Architecture, Features, and Operations

## 19.1 Positioning
TigerGraph is a graph analytics platform designed for high-performance deep-link analysis.

## 19.2 Data Model and Query
Property graph model with graph query language optimized for parallel graph computation.

## 19.3 Strengths
High-throughput graph analytics and advanced graph algorithm workloads.

## 19.4 Operational Notes
Modeling and query design should align with partition strategy and workload patterns.

## 19.5 Best Fit
Real-time fraud detection, AML, customer 360, and deep-link analytics at scale.

## 19.6 Real-Time Example
Banking platform scores suspicious account networks within strict response SLAs.
