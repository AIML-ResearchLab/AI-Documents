# 07 - Indexing, Partitioning, and Storage Architectures

## 7.1 Indexing Strategy
Index node labels, key properties, and common traversal anchors.

## 7.2 Storage Models
Native graph stores optimize pointer traversal; multi-model stores balance versatility.

## 7.3 Partitioning and Sharding
Partition by community or domain boundaries to reduce cross-shard traversals.

## 7.4 Tradeoffs
High fan-out traversals are sensitive to storage locality and network hops.

## 7.5 Real-Time Example
Recommendation engine reduces latency by co-locating user-community subgraphs.
