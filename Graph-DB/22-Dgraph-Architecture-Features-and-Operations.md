# 22 - Dgraph: Architecture, Features, and Operations

## 22.1 Positioning
Dgraph is a distributed graph database focused on horizontal scalability and low-latency queries.

## 22.2 Data Model and Query
Stores graph relations with schema-driven predicates and graph-oriented query APIs.

## 22.3 Strengths
Distributed-first design with strong performance for connected query workloads.

## 22.4 Operational Notes
Plan cluster topology, replication factors, and backup lifecycle for reliability.

## 22.5 Best Fit
Large-scale relationship applications requiring distributed graph infrastructure.

## 22.6 Real-Time Example
Ad-tech identity graph resolves cross-device relationships during campaign execution.
