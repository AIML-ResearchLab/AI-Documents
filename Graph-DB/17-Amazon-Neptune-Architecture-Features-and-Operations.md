# 17 - Amazon Neptune: Architecture, Features, and Operations

## 17.1 Positioning
Amazon Neptune is a managed graph database service on AWS.

## 17.2 Data Model and Query
Supports property graph and RDF workloads with graph query interfaces.

## 17.3 Strengths
Managed operations, AWS integration, and enterprise security controls.

## 17.4 Operational Notes
Design around instance sizing, read replicas, and workload-specific query tuning.

## 17.5 Best Fit
Teams in AWS needing managed graph infrastructure with security and scale controls.

## 17.6 Real-Time Example
Supply-chain risk graph correlates vendor, shipment, and compliance entities in AWS stack.
