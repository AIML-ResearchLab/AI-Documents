# 01 - Program Overview and MLOps Lifecycle

## 1.1 Goal
Build complete production-ready MLOps capabilities for reliable ML systems.

## 1.2 Lifecycle
Design -> data -> train -> validate -> deploy -> monitor -> improve.

## 1.3 Core Pillars
Reproducibility, reliability, governance, and continuous improvement.

## 1.4 Operating Model
Cross-functional collaboration between data science, platform, and SRE teams.

## 1.5 Real-Time Example
Fraud model updates run through automated train-validate-deploy pipeline weekly.
