# 30 - Advanced Capstone: ML System Design

## 30.1 Objective
Design a production ML system from data to monitoring with clear technical tradeoffs.

## 30.2 Required Deliverables
- architecture doc
- model/evaluation report
- deployment and rollback strategy
- monitoring dashboard specification
- governance and risk checklist

## 30.3 Rubric
- technical correctness
- robustness and safety
- operational readiness
- cost and latency efficiency
- documentation quality

## 30.4 Suggested Capstones
- fraud detection platform
- recommendation system
- support triage intelligence
- demand forecasting pipeline
