# 01 - Program Overview and A2A Ecosystem

## 1.1 Goal
Build complete practical knowledge of the Agent2Agent protocol for interoperable agent systems.

## 1.2 What A2A Enables
Standard communication between independent agents across platforms and vendors.

## 1.3 Ecosystem Components
A2A specification, agent cards, tasks, messages, artifacts, transports, and enterprise controls.

## 1.4 Real-Time Example
A travel agent delegates flight pricing to one agent and hotel booking to another through A2A.
