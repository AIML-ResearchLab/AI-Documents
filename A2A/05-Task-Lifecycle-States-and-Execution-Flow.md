# 05 - Task Lifecycle, States, and Execution Flow

## 5.1 Task Creation
Clients submit an initial message that starts a server-managed task.

## 5.2 State Model
Tasks move through lifecycle states such as submitted, working, waiting, completed, or failed.

## 5.3 Task Retrieval
Clients poll or subscribe for status and artifact updates.

## 5.4 Real-Time Example
A compliance check task pauses for human approval, then resumes and completes.
