# 13 - Building an A2A Client: Integration and Orchestration

## 13.1 Client Responsibilities
Discover agents, negotiate capabilities, submit tasks, and track lifecycle updates.

## 13.2 Orchestration Logic
Select specialized agents and compose their outputs into unified user responses.

## 13.3 Failure Handling
Implement retries, fallback agents, and graceful degradation paths.

## 13.4 Real-Time Example
A customer app routes tax queries to finance agent and legal disclaimers to policy agent.
