# 02 - A2A Core Concepts: Agents, Clients, Users, and Servers

## 2.1 Role Definitions
User-facing agents, client applications, remote agents, and service hosts each have distinct responsibilities.

## 2.2 Interaction Model
Clients discover agents, initiate tasks, exchange messages, and retrieve artifacts.

## 2.3 Collaboration Boundaries
A2A focuses on inter-agent protocol, not internal reasoning implementation.

## 2.4 Real-Time Example
A customer-support agent asks a billing-specialist agent to validate invoice discrepancies.
