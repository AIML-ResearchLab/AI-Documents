# 03 - Agent Card: Discovery, Capabilities, and Metadata

## 3.1 Agent Card Purpose
The Agent Card publishes identity, endpoint, supported skills, and protocol details.

## 3.2 Capability Advertisement
Cards declare features such as streaming, push updates, and modality support.

## 3.3 Discovery Workflows
Clients fetch and validate cards before creating task sessions.

## 3.4 Real-Time Example
A procurement agent selects only vendors whose cards declare secure file artifact support.
